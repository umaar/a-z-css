Font Display

font-display

  • Specify how a web font (@font-face) should render based on load time
  • Use a fallback font first, but cache the webfont for next page load
  • Chrome only for now

Three important portions of time to remember during web font loading

  1. The Font Block period
  2. The Font Swap period
  3. The Font Failure period

1. Font Block

Font not loaded?

Use an invisible fallback font face while waiting.

2. Font Swap

Has the web font loaded? Use it!

Still not loaded? Render with a fallback instead (from font-family)

3. Font Failure

Mark as failed. Use the fallback.

What can we do with font-display?

  • font-display: block; ~3 second block period & an infinite swap period
  • font-display: swap; no blocking at all & an infinite swap period
  • font-display: fallback; short swap & short block periods. Fallback may be used permanently

CSS Paint API

Paint custom CSS images


body {
	/*
	* This can be accessed in
	* the JavaScript paint() method
	*/
	--circle-color: green;
	background-image: paint('my-custom-circle');
}
	

registerPaint('my-custom-circle', class {
	paint(ctx, geom, properties) {
		const color = properties.get('--circle-color');
		ctx.fillStyle = color;

		// Determine the center point and radius.
		const x = geom.width / 2;
		const y = geom.height / 2;
		const radius = Math.min(x, y);

		ctx.beginPath();
		ctx.arc(x, y, radius, 0, 2 * Math.PI, false);
		ctx.fill();
	}
});

Matches

:matches

Pseudo class to group complex CSS selectors

  • Is the equivalent of selector nesting in Sass
/* Before */
section h1,
article h1,
aside h1,
nav h1 {
	color: green;
}

/* After */
:matches(section, article, aside, nav) h1 {
	color: green;
}

/* Or in Sass */
section, article, aside, nav {
	h1 {
		color: red;
	}
}

object-fit

object-fit

How an elements content reacts to the width and height of its box

  • Combine this with object-position for positioning
  • Examples include: fill, contain, cover
  • fill - aspect ratio is not preserved
  • contain - scaled up as much as possible, but only within the bounds. Aspect ratio is preserved
  • cover - scale up until the whole box is "covered", while preserving aspect ratio

root

:root

Target the highest level element in the DOM

  • Will typically be the html element
  • Useful for defining CSS custom properties here

All descendants of the root now have access to --main-color, and they can customise it if they wish.


:root {
    --main-color: #CCC;
